# Supermarket-system-cpp
A console-based supermarket system demonstrating Object-Oriented Programming concepts such as classes, structures, encapsulation and arrays. Users can perform operations as an administrator or a customer, including adding, editing removing and viewing products.
 
# Features:
## Administrator

-Login using an email with the @admin.com domain.

-Add new products with a unique code, name, price, and discount.

-Edit existing products.

-Remove products from the system.

-Return to the main menu or log out.

## Customer

-View and purchase products.

-Generate a receipt with total cost after discounts.

-Return to the main menu.

## Common

-Interactive menu system.

-Handles invalid inputs and duplicate product codes.

-Supports up to 100 products in memory.

# Technologies

-Language: C++

-Concepts used: Classes, structures, arrays, OOP, console I/O.
